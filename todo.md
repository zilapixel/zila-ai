# ZILA Token Website - Development Plan

## Design Guidelines

### Design References
- **Web3 Dashboard Style**: Dark mode with glassmorphism effects
- **Crypto Analytics**: Artemis-style data visualization
- **AI Startup Aesthetic**: Futuristic, premium feel

### Color Palette
- Primary Background: #0a0a0f (Deep Black)
- Secondary Background: #12121a (Dark Purple-Black)
- Glass Background: rgba(20, 20, 35, 0.7)
- Primary Accent: #8b5cf6 (Purple)
- Secondary Accent: #3b82f6 (Electric Blue)
- CTA Accent: #f59e0b (Neon Orange/Yellow)
- Text Primary: #ffffff
- Text Secondary: #a1a1aa
- Neon Glow: #a855f7 (Purple Glow)

### Typography
- Headings: 'Orbitron', sans-serif (Futuristic)
- Body: 'Inter', sans-serif (Clean, readable)
- Accent: 'Space Grotesk', sans-serif

### Key Component Styles
- **Glassmorphism Cards**: backdrop-filter: blur(20px), semi-transparent backgrounds, neon borders
- **Buttons**: Gradient backgrounds, glow on hover, smooth transitions
- **Navigation**: Sticky, transparent with blur, neon accents
- **Animations**: Smooth scroll, floating effects, 3D transforms
