# Project Summary
The ZILA project aims to create a futuristic Web3 website for the ZILA Token, integrating artificial intelligence, decentralized infrastructure, and meme culture. It focuses on delivering a premium user experience through a dark mode interface with neon accents, smooth animations, and a cinematic UI. The website serves as a hub for community engagement, token distribution, and showcasing the unique ecosystem of ZILA.

# Project Module Description
The ZILA website consists of several key modules:
- **Home**: Features a hero section with a 3D animated ZILA token, headline, and call-to-action buttons.
- **Ecosystem**: Displays the four pillars of ZILA's technology, including AI intelligence, Web3 infrastructure, meme distribution, and community governance.
- **Airdrop**: Allows users to pre-register for the ZILA airdrop with a modal form.
- **Tokenomics**: Provides an interactive pie chart detailing token distribution.
- **Economic Dashboard**: Displays analytics on cross-chain growth metrics.
- **Roadmap**: Outlines the development phases of the ZILA project.
- **Partnerships**: Showcases strategic partnerships with industry leaders.
- **Whitepaper**: Contains detailed documentation about ZILA's vision, tokenomics, and governance structure.
- **Footer**: Includes company and community links, security information, and a disclaimer.

# File Description Inventory
- **index.html**: The main HTML file that structures the website layout.
- **package.json**: Contains project metadata and dependencies.
- **script.js**: Manages interactivity and dynamic features of the website.
- **style.css**: Defines the visual styles and layout of the website.
- **template_config.json**: Holds configuration settings for templates used in the project.

# Technology Stack
- **HTML**: For structuring the content.
- **CSS**: For styling and layout, utilizing glassmorphism and animations.
- **JavaScript**: For interactivity and dynamic content management.
- **Chart.js**: For rendering interactive charts.
- **Web3**: For blockchain integration.

# Usage
To set up the ZILA project:
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Run `npm install` to install the dependencies.
4. Build the project using `npm run build`.
5. Start the development server to view the website.
